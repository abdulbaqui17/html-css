Quick Start

Install the Python extension from the marketplace. Pylance will be installed as an optional extension. Open a Python (.py) file and the Pylance extension will activate. Note: If you've previously set a language server and want to try Pylance, make sure you've set "python.languageServer": "Default" or "Pylance" in your settings.json file using the text editor, or using the Settings Editor UI.
